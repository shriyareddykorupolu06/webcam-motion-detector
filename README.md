# Webcam-motion-detector
Real-time motion detection using webcam with JavaScript and computer vision logic
## Features
Real-time webcame video processing
Motion detection using frame differencing
Motion status displayed on screen

## Technologies Used
HTML
JavaScript
Canvas API
WebRTC

## How to Run
Open project folder in VS Code
Right-click on index.html- Open with Live Server
Allow webcam permission

## Notes
Runs completely in the browser
Implementation and testing done by me with AI assistance
